Getting the date in Javascript

In order to get the date in Javascript you can do


const date = new Date();
console.log(date);

Which gives you the full date object and from here you can pull out specific parts like:


date.getFullYear();
date.getMonth();
date.getDate();
date.getDay();

//Or if you want a clean readable string
date.toLocaleDateString();
date.toLocaleString();
